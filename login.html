Roles y alcance

Define estos niveles en la base de datos para controlar quién puede gestionar usuarios y productos.

  • Administrador: CRUD de usuarios y CRUD de productos.
  • Supervisor: CRUD de productos de la tienda.
  • Team: Solo vista del CRUD (sin editar).
  • Usuario: Clientes que navegan y compran en la tienda.

Paso a paso: base de datos MySQL en XAMPP

Guía rápida para crear la base de datos local que soportará autenticación y permisos.

  1. Abre XAMPP Control Panel y levanta los servicios Apache y MySQL.
  2. Haz clic en Admin del módulo MySQL para entrar a phpMyAdmin.
  3. En phpMyAdmin crea una base de datos llamada ferroapp con cotejamiento utf8mb4_general_ci.
  4. Ejecuta el siguiente script SQL en la pestaña SQL para generar tablas y relaciones:
CREATE TABLE roles (
  id INT AUTO_INCREMENT PRIMARY KEY,
  nombre VARCHAR(50) NOT NULL UNIQUE,
  descripcion VARCHAR(255)
);

CREATE TABLE usuarios (
  id INT AUTO_INCREMENT PRIMARY KEY,
  nombre VARCHAR(120) NOT NULL,
  correo VARCHAR(150) NOT NULL UNIQUE,
  password_hash VARCHAR(255) NOT NULL,
  creado_en TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE usuario_rol (
  usuario_id INT NOT NULL,
  rol_id INT NOT NULL,
  PRIMARY KEY (usuario_id, rol_id),
  FOREIGN KEY (usuario_id) REFERENCES usuarios(id) ON DELETE CASCADE,
  FOREIGN KEY (rol_id) REFERENCES roles(id) ON DELETE CASCADE
);

INSERT INTO roles (nombre, descripcion) VALUES
('administrador', 'CRUD de usuarios y productos'),
('supervisor', 'CRUD de productos'),
('team', 'Solo lectura del CRUD'),
('usuario', 'Clientes de la tienda');
